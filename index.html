Manual de Segurança

  • • Backup de Dados: Como o ScaleFlow armazena dados no navegador (LocalStorage), limpe seu cache com cuidado. Salve uma cópia do arquivo index.html em um drive externo.
  • • Criptografia: Sua API Key é codificada localmente. Não compartilhe seu computador com usuários não confiáveis.
  • • Privacidade: O ScaleFlow não possui servidor central. Todos os seus dados de análise e rejeições residem apenas no seu hardware.
